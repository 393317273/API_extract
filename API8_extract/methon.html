createContext

\n 
PaintContext createContext(ColorModel cm,\n                           Rectangle deviceBounds,\n                           Rectangle2D userBounds,\n                           AffineTransform xform,\n                           RenderingHints hints)
\n
Creates and returns a PaintContext used to\n generate the color pattern.\n The arguments to this method convey additional information\n about the rendering operation that may be\n used or ignored on various implementations of the Paint interface.\n A caller must pass non- null values for all of the arguments\n except for the ColorModel argument which may be null to\n indicate that no specific ColorModel type is preferred.\n Implementations of the Paint interface are allowed to use or ignore\n any of the arguments as makes sense for their function, and are\n not constrained to use the specified ColorModel for the returned\n PaintContext , even if it is not null .\n Implementations are allowed to throw NullPointerException for\n any null argument other than the ColorModel argument,\n but are not required to do so.
\n
\n
Parameters:
\n
cm - the preferred ColorModel which represents the most convenient\n format for the caller to receive the pixel data, or null \n if there is no preference.
\n
deviceBounds - the device space bounding box\n of the graphics primitive being rendered.\n Implementations of the Paint interface\n are allowed to throw NullPointerException \n for a null deviceBounds .
\n
userBounds - the user space bounding box\n of the graphics primitive being rendered.\n Implementations of the Paint interface\n are allowed to throw NullPointerException \n for a null userBounds .
\n
xform - the AffineTransform from user\n space into device space.\n Implementations of the Paint interface\n are allowed to throw NullPointerException \n for a null xform .
\n
hints - the set of hints that the context object can use to\n choose between rendering alternatives.\n Implementations of the Paint interface\n are allowed to throw NullPointerException \n for a null hints .
\n
Returns:
\n
the PaintContext for\n generating color patterns.
